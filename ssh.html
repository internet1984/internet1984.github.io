SSH常用指令

增

rz 上传文件 touch miao 创建一个 miao的空文件，如果touch了已有的文件,则会更新那个文件的日期为当前

删

rm -R <文件名> 删除文件夹

改

mv big miao 重命名big为miao

查

df -h 查询文件大小 du -shx ./* 查询文件大小 ll 文件列表

下载

sz -ey <文件名> 就可以下载到本地了 sz -ey ./* 下载本目所有文件

MYSQL 数据库

mysqldump -u 数据库用户名 -p 数据库名 > dbooxx.sql 导出数据库为dbooxx.sql mysql -u 数据库用户名 -p 数据库名 < dbooxx.sql --default-character-set=utf8 把当前目录的 dbooxx.sql 以 utf8 编码导入,当然你原来是gbk就改成gbk 中文blog经常遇到这个情况,一不小心就 ??? 满脑子都问号了

linux操作技巧

ctrl + c 恢复命令行输入状态

VI常用指令

0或者"Home”键：光标转移到此段的最前面字节处。

$或者"End"键：光标转移到此段的最后面字节处。

n:光标向后移动n个字节。

n:光标向下移动n行。

gg:光标移动到档案第一行。

GG：光标移动到档案最后一行。

nG：光标移动到档案的第n行。

/word:向下查找word

？word:向上查找word

n：向下搜索

N：向上搜索

dd:删除光标所在的一行。

ndd:删除光标以下的n行。

yy:复制光标所在的行。

nyy:复制光标以下的n行。

p:将复制的数据在光标所在的下一行贴上

u:撤销

+r:恢复

w:保存

w!:强制保存

q:离开

q!:强制离开

：set nu :显示行号

：set nonu:取消显示行号
